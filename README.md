# 🍽️ Yulchon Lunch Archive

> “It started as a personal lunch review log during my internship.
> It became a company-wide food community.”

During my 6-month internship at Yulchon, the company provided interns with a list of 100+ restaurants available for lunch meal support.

I genuinely love food, so I started leaving detailed reviews every time I visited a restaurant — what menu was good, whether there was a waiting line, if it was worth revisiting, etc.

Later, I discovered that other interns were doing the exact same thing individually.

So I thought:

> “Why don’t we just build one place to share everything together?”

That idea became this project.

At first, it was only used among interns.
But the platform slowly spread across teams, and eventually employees across the company started using it too.

Today:

* 🍴 100+ restaurants registered
* ⭐ Around 250 community reviews
* 👥 Used beyond interns → expanded company-wide
* 💬 Real internal engagement from actual employees

This project wasn’t assigned to me.
I voluntarily designed and developed it because I wanted to improve a small everyday experience people shared together.

---

## ✨ Features

* Restaurant ranking system based on ratings & review count
* Community-written lunch reviews
* Nickname-based login (anonymous available)
* Restaurant filtering:

  * Distance
  * Price
  * Waiting time
  * Category
* CRUD support for restaurants & reviews
* Review editing/deletion permissions
* Real-time data with Supabase
* Responsive and clean UI
